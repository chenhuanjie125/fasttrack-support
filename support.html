App Overview

  • Translate selected text from the menu bar or a keyboard shortcut
  • Store copied text and images in a local clipboard history
  • Keep frequently used items in pinned clipboard slots
  • Paste what you need back into your current app quickly

System Requirements

macOS 15.0 or later.

Enable Accessibility

FastTrack uses Accessibility permission only to read text that you actively select when you trigger translation.

  1. Open FastTrack Settings.
  2. Click the button to enable Accessibility access.
  3. In macOS System Settings, allow FastTrack under Privacy & Security > Accessibility.

Enable Automation for Shortcuts Events

FastTrack uses Apple's Shortcuts Events process to run its built-in system translation shortcut.

  1. Open macOS System Settings.
  2. Go to Privacy & Security > Automation.
  3. Allow FastTrack to control Shortcuts Events.

Purchase Help

The Mac App Store version offers a one-time full unlock purchase.

  1. Open the purchase screen in FastTrack.
  2. Click Restore Purchases.
  3. Make sure you are signed in with the same Apple ID used for the purchase.

Clipboard Notes

Clipboard history is stored locally on your Mac. You can disable capture or clear unlocked history from the app settings.

Feedback

When contacting support, include your macOS version, FastTrack app version, a short description of the issue, and a screenshot if the issue is visual.
