Overview

What FastTrack provides

FastTrack for macOS is built around two core tools: fast translation and clipboard management. It lets you trigger translation from the menu bar or a keyboard shortcut, while also managing local clipboard history, pinned slots, and quick paste workflows.

Quick Start

Basic workflow

  1. Launch FastTrack and open the main menu from the menu bar icon.
  2. Before using translation for the first time, verify that the built-in system translation shortcut is installed and grant the required permissions in Settings.
  3. Select text in any app, then choose Translate Current Selection from the menu bar or use the configured shortcut.
  4. After copying text or images, open the clipboard panel to review recent history and pinned slots.

Permissions

Accessibility and Automation

Accessibility is used only to read the currently selected text in another app when you explicitly trigger translation.

  1. Open FastTrack Settings.
  2. Use the in-app button to open System Settings.
  3. Allow FastTrack under System Settings > Privacy & Security > Accessibility.

Automation / Apple Events is used only to communicate with Apple’s Shortcuts Events process in order to run the built-in system translation shortcut.

  1. Open System Settings > Privacy & Security > Automation.
  2. Allow FastTrack to control Shortcuts Events.

If you deny these permissions, FastTrack can still launch, but selection-based translation features will not work correctly.

Purchases

Unlock and restore

The Mac App Store version offers a one-time unlock called Fast track Pro. It removes translation limits and unlocks all clipboard pinned slots.

  1. Open the purchase screen inside FastTrack.
  2. If the unlock is not active, click Restore Purchases.
  3. Make sure you are signed in with the same Apple ID used for the purchase.

If restore still fails, email support with your macOS version, app version, a screenshot of the purchase screen, and a short problem description.

Clipboard and Local Data

What stays on device

FastTrack keeps clipboard history and translation history on your Mac only. This data is not synced to a developer-operated server.

  • You can disable clipboard capture in Settings.
  • You can remove individual clipboard entries or clear unlocked history.
  • Translation history is stored locally in the app support directory for the history window.

Frequently Asked Questions

Common problems

Selected text does not translate
Check Accessibility and Automation permissions, and verify that the built-in system translation shortcut has been installed.

Automation error mentioning Shortcuts Events
Open System Settings > Privacy & Security > Automation and allow FastTrack to control Shortcuts Events.

Clipboard capture is not recording new items
Check whether clipboard capture is disabled, and make sure the copy action happened while FastTrack was running.

Purchase completed but features are still locked
Use Restore Purchases on the purchase screen and relaunch the app if needed.

Contact Support

How to reach us

For support, contact fasttrack@showleafy.com and include the following where possible:

  • Your macOS version
  • Your FastTrack version
  • A short description of the issue and reproduction steps
  • A screenshot or screen recording if the issue is visual
